Medium Well, a clone of Medium.com, is a fake news site where users can read fake news articles, and interact with stories and other users via leaving comments and likes. Upon landing on the site a user is met with the home page. Whether logged in or not, a user is able to view the stories on the site.

Upon clicking on either of the links in the NavBar, the user is directed to either the sign in or get started page. Each of these pages, along with the NavBar have been aptly styled to closely match the design of the original site.

Upon clicking on a story, the user is brought to a new page featuring a full sized image and the storyâ€™s contents.

After reading a story, a logged-in user may leave a comment or like the story. This functionality was implemented using AJAX on multiple backend POST route calls made to the postgreSQL database.
