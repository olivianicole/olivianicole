PlantMe is a site designed for users to buy and sell plants! Upon visiting the site a user is met with the splash page, shown on the left. The ‘Register’ and ‘Log In” buttons open up modals upon being clicked. After registering or logging in the user is brought to a homepage.

The homepage features a personalized welcome message, suggested listings, and a feed of plant listings from various sellers. The site’s NavBar has a search bar that allows a user to search any of the plants in the database, via a function on the front-end that filters through a query on every keystroke. Additionally, the user can view their cart, their favorited items, and their account page.

When a user visitors their account page they can click between three different tabs: favorites, shop details, and reviews. If a user has created a shop, the shop details tab is where they can view the listings already in their shop, as well as add additional listings. If a user has not created a shop, a form appears for the user to create one! The reviews tab is populated upon a user making a purchase. Only after a user has purchased a plant can they leave a review on the listing.

After navigating to a specific listing, a user can select a quantity and add the plant to their cart, as shown on the left. Items can also be deleted from the cart, and when you’re ready to checkout simply click “Proceed to checkout” to be directed to the checkout page!
